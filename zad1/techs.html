Czym są technologie webowe?

Krótko mówiąc, jest to grupa technologii używana do programowania aplikacji, serwisów czy też stron internetowym.

Do tej grupy należą zarówno języki skryptowe, tzw. client-side takie jak HTML, CSS czy też JavaScript, jak i technologie server-side,których może być o wiele więcej.

Poniżej opisane są w dużym skrócie najważniejsze technologie webowe:

HTML, czyli Hyper-Text Markup Language to język oparty na znacznikach, opisowy. Przechowuje on zawartość widoku strony internetowej

CSS, Cascading Style Sheets definiuje styl elementów strony, tzn nadaje im odpowiedni wygląd.

JavaScript to język programowania skryptowego stosowany głównie do obsługi interakcji użytkownika z widokiem.

PHP to także język programowania skryptowego, aczkolwiek używany głównie do budowania aplikacji internetowych po stronie serwera.

ASP .NET to zbiór technologii opartych na frameworku .NET przeznaczony właśnie do budowy aplikacji internetowych
